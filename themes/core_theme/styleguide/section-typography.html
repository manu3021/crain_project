5.1 #Typography.1 Class Used to Headings And for link

.h1 , .h2, .h3, .h4, .h5, .intro-text are all heading classes if you want to use style of heading on any tag use those classes.

Parameters:
  • H1, .h1
    Font-Size 45px; Font-Family:Roboto Black;Font-Weight: 900;
  • H2, .h2
    Font-Size 38px; Font-Family:Roboto Black;Font-Weight: 900;
  • H3, .h3
    Font-Size 24px; Font-Family:Roboto Bold;;Font-Weight: 900; Text-Transform: Uppercase
  • H4, .h4,.intro-text
    Font-Size 20px; Font-Family:Roboto Regular; Font-Weight: 400;
  • H5, .h5
    Font-Size 18px; Font-Family:Roboto Bold; Font-Weight: 400;Text-Transform: Uppercase;
  • .link
    This class is used for link
  • p
    Font-Size: 18px; Font-Weight: 300;
