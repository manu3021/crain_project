6.3 #Typography.3 Mobile

.h1 , .h2, .h3, .h4, .h5, .intro-text are all heading classes if you want to use style of heading on any tag use those classes.

Parameters:
  • H1, .h1
    Font-Size 29px; Font-Family:Roboto Black;Font-Weight: 900;
  • H2, .h2
    Font-Size 22px; Font-Family:Roboto Black;Font-Weight: 900;
  • H3, .h3
    Font-Size 18px; Font-Family:Roboto Bold;;Font-Weight: 900; Text-Transform: Uppercase
  • H4, .h4,.intro-text
    Font-Size 16px; Font-Family:Roboto Regular; Font-Weight: 400;
  • H5, .h5
    Font-Size 14px; Font-Family:Roboto Bold; Font-Weight: 700;Text-Transform: Uppercase;
  • .link
    This class is used for link
  • p
    Font-Size: 14px; Font-Weight: 300;
